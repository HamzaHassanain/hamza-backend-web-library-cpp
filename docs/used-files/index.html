Blog API Documentation

Welcome to the Blog API built with HH Web Framework!

Public Endpoints

  • GET /api/blogs - Get all published blog posts
  • GET /api/blogs/:id - Get a specific published blog post

Authentication

  • POST /api/auth/login - Admin login (username: admin, password: admin)
  • POST /api/auth/logout - Admin logout

Admin Endpoints (Requires Authentication)

  • GET /api/admin/blogs - Get all blog posts (including unpublished)
  • POST /api/admin/blogs - Create a new blog post
  • PUT /api/admin/blogs/:id - Update a blog post
  • DELETE /api/admin/blogs/:id - Delete a blog post

Example Login Request:

POST /api/auth/login
{
    "username": "admin",
    "password": "admin"
}

Example Blog Creation:

POST /api/admin/blogs
Authorization: Bearer YOUR_TOKEN
{
    "title": "My First Blog",
    "content": "This is the content of my blog post...",
    "author": "Admin",
    "published": true
}

Use the token from login response in Authorization header for admin endpoints.
