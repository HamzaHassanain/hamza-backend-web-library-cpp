CRUD API Server

Welcome to the Items CRUD API Server! Use the following endpoints to interact with items:

API Endpoints

GET /api/items

Get all items

GET /api/items/:id

Get a specific item by ID

POST /api/items

Create a new item

{
  "name": "Item name",
  "description": "Item description",
  "price": 19.99
}
PUT /api/items/:id

Update an existing item

{
  "name": "Updated name",
  "description": "Updated description",
  "price": 29.99
}
DELETE /api/items/:id

Delete an item
